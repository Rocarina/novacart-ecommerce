# 🛒 NovaCart - Mini E-Commerce Store

A Full Stack Mini E-Commerce Store built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and JWT Authentication.

## 📌 Features

- User Registration & Login
- JWT Authentication
- Product Listing
- Product Search
- Category Filter
- Shopping Cart
- Increase / Decrease Quantity
- Remove Items from Cart
- Checkout System
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcryptjs
